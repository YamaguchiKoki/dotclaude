## 出力ルール

1. ハッシュは `git log --oneline` で確認した実際のコミットハッシュを記述する
2. コミットメッセージは `fix: #番号 ...` 形式で記述する
3. 変更ファイル数は `git show --stat {hash}` で確認した値を記述する
4. 対応する受け入れ条件 # は `issue-plan.md` の番号と一致させる
5. コミットが存在しない場合（コミット禁止環境など）は理由を明記する
